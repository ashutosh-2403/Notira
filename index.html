§ 01 Features
Every
action.
Synced.

The moment anything changes in Jira — Notira reflects it in Notion. Instantly. Automatically. Without touching anything.

Notira treats your Notion page as a living document — not a one-time snapshot. Every field. Every change. Every comment. Tracked.

F-01
Real-time Webhook Sync
Jira fires a webhook. Notira processes it instantly. HMAC-SHA256 signature verification blocks fake requests.
F-02
In-place Block Updates
Description changes patch the actual Notion block directly. Not a new page. The real content updates live.
F-03
Live Stage Tracker
Backlog → In Progress → In Review → Done. Updates the moment you move a card in Jira.
F-04
Collapsible Change History
Every field change logged with author and timestamp in collapsible Notion toggle blocks.
F-05
Comment Sync
Jira comments append to the Notion page automatically with author name and timestamp.
F-06
AI Summary
Auto-generated AI summary callout on every Notion page. Instant context without reading the full description.
F-07
Redis Event Queue
Async queue with dead letter support. Events never lost. Failed syncs retry 3× with exponential backoff.
F-08
One Command Deploy
Self-hosted via Docker Compose. Add API keys to .env, run docker compose up. Done.
F-09
Your Data. Your Server.
Nothing leaves your infrastructure. Full audit log in your own PostgreSQL database. MIT licensed.
§ 02 How It Works
Five steps.
Fully automated.
01
Jira fires webhook
Any ticket event triggers a POST request to your server
02
Signature verified
HMAC-SHA256 confirms it's genuinely from Jira
03
Event queued
Saved to PostgreSQL and pushed to Redis async queue
04
Notion updated
Page created or blocks patched via Notion Blocks API
05
History logged
Change logged with author and timestamp in toggle block

§ 04 Quick Start
Up in 60
seconds.
Three commands.
You're live.
# 1. Clone
git clone github.com/ashutosh-2403/notira
cd notira

# 2. Configure
cp .env.example .env
# edit .env with your keys

# 3. Run
docker compose up -d

# Register Jira webhook URL:
# your-server/api/webhooks/jira

# Done. ✓
Configure your .env
JIRA_BASE_URL
Your Atlassian workspace URL
JIRA_EMAIL
Your Atlassian account email
JIRA_API_TOKEN
API token from id.atlassian.com/security/api-tokens
JIRA_WEBHOOK_SECRET
Any random string for HMAC verification
NOTION_API_TOKEN
Integration token from notion.so/my-integrations
NOTION_DATABASE_ID
Your Notion database ID from the page URL
